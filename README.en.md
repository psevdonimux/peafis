**p** - practical

Minimalist start page with quick search, weather and hidden labels system.

## Features

- Search via Google, Yandex, Bing, DuckDuckGo, Perplexity
- Quick access to ChatGPT
- Weather widget for 200+ cities
- Custom wallpapers (import/export/random)
- Hidden labels system for quick link access

## Hidden Labels

Create interactive links anywhere on the screen.

### Controls

| Action | PC | Phone |
|--------|:--:|:-----:|
| Labels mode | Middle click | Two-finger tap |
| Create label | Long click (500ms) | Long tap (250ms) |
| Open link | Click | Tap |
| Edit label | RMB / long click | Long tap |

## Command Line (CLICK)

| Command | Description |
|---------|-------------|
| `list` | Show saved links |
| `copy` | Copy to clipboard |
| `key url` | Add link |
| `delete key` | Delete link |
| `clearAll` | Clear all |
